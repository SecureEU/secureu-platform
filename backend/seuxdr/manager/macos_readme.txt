# SEUXDR Installation Guide (macOS)

SEUXDR is a host-based intrusion detection system (HIDS) that monitors logs, ensures file integrity, and actively responds to security threats.

## 📥 Installation on macOS

To install SEUXDR, follow these steps:

1. Move the downloaded folder to the Applications folder
   
2. Double click to run the seuxdr_installer

3. Enter your Administrator password

4. Done!


## To remove SEUXDR from your Mac:

1. Double click to run the seuxdr_uninstaller

2. Enter your administrator password

3. Done!
