# SEUXDR Installation Guide (Windows)

SEUXDR is a host-based intrusion detection system (HIDS) that monitors logs, ensures file integrity, and actively responds to security threats.

## 📥 Installation on Windows

1. **Run the installer as Administrator**  
   - Right-click `install_seuxdr_windows.exe`  
   - Select **"Run as administrator"**  
   - Follow the on-screen instructions  

## 🛑 Uninstallation

1. **Run the uninstaller as Administrator**  
   - Right-click `uninstall_seuxdr_windows.exe`  
   - Select **"Run as administrator"**
