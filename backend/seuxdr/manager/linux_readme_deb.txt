# SEUXDR Installation Guide (Debian-based Systems)

SEUXDR is a host-based intrusion detection system (HIDS) that monitors logs, ensures file integrity, and actively responds to security threats.

## 📥 Installation (Debian-based Linux)

To install SEUXDR on Debian, Ubuntu, or any Debian-based distribution, follow these steps:

```bash
sudo dpkg -i seuxdr_CloneSystems_1_arm64.deb

To remove SEUXDR from your system, use:

```bash
sudo dpkg --remove seuxdr
